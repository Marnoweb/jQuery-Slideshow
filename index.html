jQuery Slideshow

jQuery Slideshow is a performant and flexible slideshow/carousel plugin with support for different (and extendable) transitions and swipe left/right touch gestures.

Example

// Create slideshow instances
var $s = $('.slideshow').slides(opts);

// Access an instances API
var api = $s.eq(0).data('slides');

Options

Options can be specified via the standard jQuery plugin interface or as data attributes on the target element itself.

auto
Autoplay timeout in milliseconds. Set to false for no autoplay. Default: 6000.
speed
Animation speed between slides in milliseconds. Default: 600.
carousel
Selector for the carousel element. Default: .carousel.
items
Selector for carousel items. Default: .slide.
easing
Animation easing for single transition. Default: swing.
easeIn
Animation easing on fade in. Default: swing.
easeOut
Animation easing on fade out. Default: swing.
pagination
Render pagination. Default: true.
skip
Render next/previous skip buttons. Default: true.
queue
Allow .to() method while animations are queued. Default: false.
loop
Allow slideshow to loop. Default: false.
transition
Specify transition. Default: scroll.
gestures
Allow touch swipe events to control previous/next. Default: true.
onupdate
A callback function to execute on update event. Default: false.

Public methods

.hasNext()
Are there any slides after current item (ignores loop). Returns boolean.
.hasPrevious()
Are there any slides previous to current item (ignores loop). Returns boolean.
.next()
Go to the next slide.
.previous()
Go to previous slide.
.to(index)
Go to slide. Index Slide position.
.redraw([transition])
Redraw the carousel. Transition New transition style.
